title: Setup
---

For this exercise, you will need to:

- Download the [sample data](data/BBBC021_data.zip) and unzip the archive.
- If you have not already, download and install CellProfiler. You can download
CellProfiler from here:
https://cellprofiler.org/releases

If you have any trouble with these steps, please let one of the teachers know
so they can help you set this up before proceeding. Once you are ready, navigate
to the exercise using the arrow buttons below.
